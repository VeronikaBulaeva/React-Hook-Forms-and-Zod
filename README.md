# Карточка пользователя, состоящая из 2х табов.

- Контактная информация
- Проекты 

## Стек:

- React
- TypeScript
- react-hook-form
- yup
- mui
- eslint, prettier

  
## Требования:

- В заголовке увеличивается счетчик проекта, т.е. "Проект 1", "Проект 2", ..., "Проект n"
- Каждый новый проект валидируется при клике на кнопку "Добавить". Валидация полей происходит только после первого нажатия на "Добавить". 
- Дата начала работы не может быть позже, чем дата окончания работы. Дата окончания работы может быть не заполнена. 
- Максимальное количество проектов в строке - 2, остальные переносятся на следующую строку. 
- При клике на "Добавить" поля формы становятся ReadOnly, убирается возможность редактирования. Также скрывается кнопка удаления проекта и добавления, появляется кнопка "Редактировать"
- После нажатия на кнопку "Сохранить" все поля формы переходят в readOnly-режим, появляется кнопка "Редактировать". При нажатии на кнопку "Сохранить" происходит валидация кнопки. Таб с некорректными данными подсвечивается красным.
